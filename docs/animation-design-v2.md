# diraaax Animation Design v2

Animations must feel:

soft
romantic
dreamy

Avoid fast or aggressive animations.

---

# Floating Hearts

Used in:

- login page
- capsule pages
- hero sections

Small hearts float slowly upward.

---

# Polaroid Drop

Used in gallery and memory pages.

Animation:

photos fall from top  
slightly rotate  
settle into grid

---

# Capsule Opening

Sequence:

1. capsule card appears
2. envelope opens
3. heart particles appear
4. memories slideshow begins

---

# Memory Reveal

Photos fade in with soft upward motion.

Timing:

300–500ms

---

# Message Reveal

Love letters appear line by line.

Use:

typewriter effect or fade-in text.

---

# Confetti Hearts

Used when:

- completing bucket list item
- opening capsule

Small pink heart particles burst briefly.
