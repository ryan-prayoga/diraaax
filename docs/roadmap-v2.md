# diraaax v2 Roadmap

This document defines the feature roadmap for diraaax v2.

The goal is to transform diraaax into a **romantic digital memory universe**.

---

# Phase 1 — Core Memory Experience

Features:

- Relationship Timeline
- Anniversary Counter
- Memory of the Day
- Polaroid Gallery UI
- Bucket List Upgrade

---

# Phase 2 — Emotional Features

Features:

Love Letter Capsules

Users can write messages that unlock in the future.

Example:

Open in: Jan 1 2027

When opened, the capsule launches a **special animated love page**.

Additional features:

- animated memory slideshow
- love message reveal
- background music

---

# Phase 3 — Couple Interaction

Features:

Mood Tracker

Example:

Ryan: 😊  
Dira: 😴

Things I Love About You

Example:

- You support me in ML
- Your smile
- Your cute photos

Voice Messages

Users can leave voice notes for each other.

---

# Phase 4 — Nostalgia & Map

Features:

Map of Memories

Pins:

- first meet
- first date
- travel memories

Each pin opens a memory card.

---

# Phase 5 — Advanced

Future features:

- AI Nostalgia Assistant
- Push notifications
- PWA install
- relationship statistics

---

# Final Goal

diraaax becomes a **living digital space of Ryan & Dira's relationship**.
